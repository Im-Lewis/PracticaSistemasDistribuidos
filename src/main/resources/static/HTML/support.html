FREQUENT QUESTIONS
In the case where you want to buy a shovel or other object, how can you do it?
In order to buy a product, there are several options:
  1. Search directly for the name of the product that we want to buy in the search container. Select the product and add it to the cart to then make the purchase.
  2. Go to the "Shop" option in the drop-down menu at the top, search for the product using the filters we offer and carry out the same purchase operation as in the previous section
Can I buy a product without having to create an account with my credentials?
No. We need to have data about the person who wants to buy the product for easier transfer and contact with the user in the event of a possible incident
In the tournaments section, if I want to sign up for a tournament, how do I see which tournament I'm signed up for?
In order to see which tournaments you are signed up for, you just need to select the icon that is displayed at the bottom right of the screen to see all that information
